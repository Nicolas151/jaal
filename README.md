# JAAL Rituel Beauté

Site web statique complet pour la marque JAAL, prêt à être publié sur GitHub Pages.

## Structure

- `index.html` : page principale du site
- `style.css` : styles, mise en page responsive et direction artistique
- `script.js` : menu mobile, animations discrètes, bouton retour en haut et formulaire email
- `assets/img/` : logo et visuels optimisés pour le web
- `Infos/` et `Photos/` : sources fournies, conservées comme fichiers de référence

## Publication GitHub Pages

1. Pousser ce dossier dans un dépôt GitHub.
2. Aller dans `Settings` > `Pages`.
3. Dans `Build and deployment`, choisir `Deploy from a branch`.
4. Sélectionner la branche `main` et le dossier `/root`.
5. Enregistrer. Le site sera publié automatiquement par GitHub Pages.
